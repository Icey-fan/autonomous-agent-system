# 🤖 Autonomous Agent System

具备**复杂任务拆解、长链推理、多智能体协同与闭环校验**能力的自主Agent系统。

## ✨ 核心特性

- **🧩 复杂任务拆解**：自动分解为树形子任务，支持顺序/并行/条件分支
- **🧠 长链推理**：多步逻辑推理链 + 业务规则引擎 + 不确定性处理
- **👥 多智能体协同**：6个专用智能体并行协作
- **🔄 闭环校验**：6维度质量检查，失败自动重试（指数退避）
- **📊 决策输出**：风险评估 + 备选方案 + 行动项 + 决策追溯
